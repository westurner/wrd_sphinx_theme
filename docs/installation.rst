============
Installation
============


Stable release
--------------

To install WRD Sphinx Theme, run this command in your terminal:

.. code-block:: console

    $ pip install wrd_sphinx_theme

This is the preferred method to install WRD Sphinx Theme, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for WRD Sphinx Theme can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/westurner/wrd_sphinx_theme

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/westurner/wrd_sphinx_theme/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/westurner/wrd_sphinx_theme
.. _tarball: https://github.com/westurner/wrd_sphinx_theme/tarball/master
